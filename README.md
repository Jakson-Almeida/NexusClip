# NexusClip

Plataforma para criar e gerenciar canais de cortes em múltiplas contas e plataformas.

## Sobre

NexusClip centraliza o fluxo de trabalho de cortes: da detecção de momentos em vídeos longos até a publicação em redes sociais, com suporte a múltiplas contas por usuário.

## Stack

- **Frontend:** React + TypeScript + Vite
- **Backend:** em breve

## Desenvolvimento

```bash
npm install
npm run dev
```

O app estará disponível em `http://localhost:5173`.

## Scripts

| Comando         | Descrição              |
| --------------- | ---------------------- |
| `npm run dev`   | Servidor de desenvolvimento |
| `npm run build` | Build de produção      |
| `npm run preview` | Preview do build     |
